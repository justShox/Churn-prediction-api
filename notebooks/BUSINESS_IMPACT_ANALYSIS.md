# Анализ бизнес-эффекта модели предсказания оттока

## 1. Baseline vs модель

### Без модели (текущая ситуация)
- Всего клиентов: 15,000
- Ушедших клиентов: 3,058 (20.4%)
- Все 3,058 = потеря для банка

### С моделью (CatBoost tuned, threshold=0.5)
- Recall = 84.80% → правильно ловим 2,593 из 3,058
- Пропущенные (FN) = 465 клиентов (15.2%)
- Удержано моделью = 2,593 клиентов

### Воронка оттока
```
Все клиенты:        15,000 (100%)
Ушли без модели:     3,058 (20.4%)
Ушли с моделью:        465 (3.10%)  ← пропущенные моделью
Удержано моделью:    2,593 (17.3%)
Остались (TP+TN):  14,535 (96.9%)
```

### Ключевые пороги:
- **Лучший по KS:** threshold = 0.44 (KS = 0.719)
- **Лучший по F1:** threshold = 0.70 (F1 = 0.741)
- **Дефолтный:** threshold = 0.50 (F1 = 0.712)

### Как выбирать порог:
- **Банк консервативный** (дешевле ловить, дороже пропустить) → threshold ниже (0.3-0.4)
- **Банк агрессивный** (дорогие retention-кампании, не хочется тратить на FP) → threshold выше (0.6-0.7)
- **Баланс** → текущий 0.5 или оптимальный 0.44

## 3. ROI-анализ (примерный)

### Входные данные (примерные):
- Средний revenue на клиента в год: 50,000 тенге
- Стоимость retention-кампании на клиента: 5,000 тенге
- Конверсия retention (клиент остался после кампании): 40%

### При threshold=0.44 (оптимальный по KS):
- Контактов (TP + FP) = 540 + 389 = 929
- Стоимость кампании = 929 × 5,000 = 4,645,000 тенге
- Удержано (из TP) = 540 × 0.40 = 216 клиентов
- Выручка удержанных = 216 × 50,000 = 10,800,000 тенге
- **Чистая выгода = 10,800,000 − 4,645,000 = 6,155,000 тенге**
- **ROI = 132%**

### При threshold=0.50 (текущий):
- Контактов (TP + FP) = 519 + 326 = 845
- Стоимость кампании = 845 × 5,000 = 4,225,000 тенге
- Удержано (из TP) = 519 × 0.40 = 208 клиентов
- Выручка удержанных = 208 × 50,000 = 10,400,000 тенге
- **Чистая выгода = 10,400,000 − 4,225,000 = 6,175,000 тенге**
- **ROI = 146%**

### При threshold=0.70 (оптимальный по F1):
- Контактов = 446 + 145 = 591
- Стоимость кампании = 591 × 5,000 = 2,955,000 тенге
- Удержано = 446 × 0.40 = 178 клиентов
- Выручка = 178 × 50,000 = 8,900,000 тенге
- **Чистая выгода = 8,900,000 − 2,955,000 = 5,945,000 тенге**
- **ROI = 201%**

---

## 4. Выводы

1. **Модель окупается** — ROI > 100% при любом разумном пороге
2. **Оптимальный порог** зависит от бизнес-стратегии:
   - Если retention дешёвый → снижать порог (ловить больше)
   - Если retention дорогой → повышать порог (ловить точнее)
3. **Текущий порог 0.5** — хороший баланс (ROI ~146%)
4. **Порог 0.7** — максимальный ROI (~201%), но пропускаем 27% оттока
5. **Главный риск** — FP (ложная тревога): тратим бюджет на клиентов, которые не собирались уходить. Но это лучше, чем пропустить реальный отток.
